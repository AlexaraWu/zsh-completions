# zsh-completions

my zsh completion functions.

## Command list

- `conda`
- `diskutil`
- `p7zip`
  - `7z`
  - `7za`
  - `7zr`
- `pandoc`
- `you-get`

## Usage

Adding into `.zshrc`:

```shell
source path/to/zsh-completions/zsh-completions.plugin.zsh
```
